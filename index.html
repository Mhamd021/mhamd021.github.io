Waitless - Digital Queue System

  • NestJS
  • Redis
  • BullMQ
  • Socket.IO
  • PostgreSQL
  • Next.js

Live queue management platform replacing physical waiting lines with QR/link-based digital queues.

  • WebSockets push position updates only on real state changes.
  • BullMQ keeps email notifications asynchronous and restart-safe.
  • Customer access through QR/link tokens without accounts.
  • MongoDB record varaible-shape events for analytics
GitHub
Real-time Queue Management Customers join via a link: no app, no login required. Position updates instantly.
BullMQ + Redis Email notifications are queued as restart-safe background jobs.
Socket.IO Real-time All connected clients update instantly when admin calls next, with no polling needed.
JWT + Secret Key Auth Admin is protected by JWT. Registration requires a secret key, so there is no open sign-up.
Customer Token No login for customers. A UUID token acts as their personal tracking link.
MongoDB MongoDB as a dedicated event log. each queue action written as a document with flexible per-event metadata, then aggregated via MongoDB pipeline to produce live no-show rate, completion rate, and drop-off rate per queue.
Docker Compose PostgreSQL and Redis run in containers, so one command starts the full stack.
